# Utils Layout

This folder is organized by helper domain. New helpers should be added to the matching file.

## Domain mapping

- `HTTP_connect.cpp`: HTTP request helpers
- `json_utils.cpp`: JSON parse/serialize helpers
- `string_utils.cpp`: generic string/text conversion and normalization helpers
- `command_utils.cpp`: command matching, prefix parsing, and route dispatch helpers
- `number.cpp`: numeric conversion helpers
- `random.cpp`: random engine and number generation
- `file_utils.cpp`: file read/write/open and path helpers
- `qq_utils.cpp`: QQ/group identity helpers
- `image_utils.cpp`: image and download helpers
- `uuid_utils.cpp`: UUID helpers
- `log.cpp`: logging helpers
